# Vector OS — Engineering Docs

Engineering documentation for the Vector OS monorepo.

## UI Lab

Component development happens in Storybook. Run from the repo root:

```bash
pnpm dev:storybook
```

## Packages

| Package | Purpose |
|---------|---------|
| `@vector-platform/tokens` | Design tokens (colours, typography, spacing, radius, shadows, motion) |
| `@vector-platform/ui` | Reusable UI components |
| `@vector-platform/icons` | Icon components |
| `@vector-platform/utils` | Shared utilities |
| `@vector-platform/types` | Shared TypeScript types |
