# Vector OS — Engineering Docs

Engineering documentation for the Vector OS monorepo.

## UI Lab

Component development happens in Storybook. Run from the repo root:

```bash
pnpm dev:storybook
```

## Packages

| Package | Purpose |
|---------|---------|
| `@vector-os/design-system` | Design tokens (colours, typography, spacing, radius, shadows, motion) |
| `@vector-os/ui` | Reusable UI components |
| `@vector-os/icons` | Icon components |
| `@vector-os/utils` | Shared utilities |
| `@vector-os/types` | Shared TypeScript types |
