# MilePilot Design Review

Every new screen must pass these five questions before we lock it.

1. **Would I happily use this every day?**
2. **Does this feel calm and uncluttered?**
3. **Can I understand it in under three seconds?**
4. **Does it match the MilePilot design language?**
5. **Would I be proud to show this in an App Store screenshot?**

If the answer to any question is **no**, keep refining.

If the answer to all five is **yes**, lock the screen and move on.

## Design language (benchmark)

- Dark onboard gradient with soft blue glow
- Brand bar: wordmark + living pulse underline
- Brand signature: *Your business. On auto pilot.* — glowing line + tagline
- Conversation over forms — one screen, one idea
- Premium blue CTAs with calm hover/tap feedback
- Generous spacing — nothing shouting
