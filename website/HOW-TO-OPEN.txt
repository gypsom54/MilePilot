MilePilot Website — how to open
================================

1. Download MilePilot-WEBSITE-v4.6-DOWNLOAD.zip from GitHub
2. Unzip to a NEW folder (not inside an old copy)
3. Double-click OPEN-WEBSITE.bat

The black window should say "Opening website..." then stay open.
Your browser should open index.html automatically.

If the browser does not open:
- Double-click index.html in the same folder
- Or drag index.html into Chrome, Edge or Firefox

If you see "index.html was not found":
- You are in the wrong folder
- Unzip the download zip again to a fresh folder
- Run OPEN-WEBSITE.bat from inside that folder (next to index.html)

If the page looks old:
- Clear browser cache or use a private/incognito window
