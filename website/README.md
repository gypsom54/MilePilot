# MilePilot Website v1.3

Premium product site — the app brought to life. Navy panels, electric blue GPS pulse, AutoPilot brand moment.

**Live targets:** `milepilot.uk` (marketing) · `app.milepilot.uk` (product)

---

## Deploy (Cloudflare Pages)

1. Create a Cloudflare Pages project (or use existing `milepilot` site)
2. Upload the **`website/`** folder contents (not the folder itself)
3. Required files: `index.html`, `styles.css`, `main.js`
4. Custom domain: `milepilot.uk` or `www.milepilot.uk`
5. Verify homepage loads and CTAs point to `https://app.milepilot.uk`
