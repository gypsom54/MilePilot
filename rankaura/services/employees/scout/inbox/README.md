# Opportunity Inbox

Holds Scout-discovered opportunities awaiting business owner review.

## Statuses

| Status | Meaning |
| ------ | ------- |
| `new` | Just discovered |
| `reviewing` | Under consideration |
| `accepted` | Approved for action (via AuraCore) |
| `dismissed` | Not pursuing |

## Phase 1

In-memory mock inbox. Maps to dashboard Opportunities section.

## AuraCore connection

Accept/dismiss actions route through AuraCore in Phase 2.
