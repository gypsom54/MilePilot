# Opportunity Scoring Engine

Ranks Scout opportunities by confidence, visitor potential, and lead potential.

## Scoring formula

Weighted sum (Phase 1):

- Confidence: 40%
- Estimated visitors: 30%
- Potential leads: 30%

## Priority thresholds

| Score | Priority |
| ----- | -------- |
| 80+   | high     |
| 60–79 | medium   |
| <60   | low      |

## Phase 1

Deterministic mock scoring. No machine learning.
