# Research Reports

Scout generates three report types for business owners.

## Report types

| Period | Purpose |
| ------ | ------- |
| Daily | Quick briefing on today's discoveries |
| Weekly | Week-in-review with trends and opportunities |
| Monthly | Comprehensive growth research summary |

## Phase 1

Template-based mock reports. No AI generation.

## Dashboard connection

Daily report feeds the hero brief. Opportunities feed the inbox.
